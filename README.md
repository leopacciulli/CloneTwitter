<h2 align="center">:rocket: Layout Twitter :rocket:</h2>

### :hammer_and_wrench: Projeto:
- O projeto desenvolvido com html e css puro junto a uma aula assistida da Rocketseat onde foi desenvolvido o layout do Twitter, sendo a construção de uma tela estática, com foco no ensino do #Flexbox.

- Porém, aumentei meus desafios. Construindo a aplicação utilizando React Native e Styled-components para estilização da página. E por se tratar de um layout antigo, fiz também no mesmo projeto um outro layout, mostrando o novo visual do Twitter em como ele está atualmente.

- Fiz com a ajuda do css/transition a funcionalidade para alteração dos layouts apenas clicando em um botão no canto superior esquerdo da página.
